Задание от 2019.10.25

К этому моменту подразумевается что вы прочитали статьи по es6 и npm

з.ы. Эта та самая домашка, по которой рекомендуется спрашивать вопросы, будем разбираться, если не всё понятно.

  • Добавить package.json файл через npm init в проект
  • Установить babel-cli.
  • Добавить следующие команды в package.json
    • npm run build:js для компиляции js вашего проекта.
    • npm run dev:js для watch режима js изменений
    • npm run build:css для компиляции sass вашего проекта.
    • npm run dev:css для watch режима css изменений.
  • Добавить .gitignore файл и поместить туда node_modules и файлы компиляции css/js, чтобы они более не появлялись в вашем репозитории
  • Скомпилировать проект и убедиться что ваш код соответствует современному стандарту языка
    • Не использовать var
    • Использовать стрелочные функции, вместо function
    • (опционально и забегая вперёд) Добавить класс App (или структуру классов, если вы знакомы с ООП, тогда возможно вам понадобится import/export конструкции, которые стали доступны благодаря es6 babel) для описания консольного квиза. Например (на ваше усмотрение) должны быть методы run(), complete(). Также должны быть хранимые свойства внутри класса касательно вопросов квиза, текущего вопроса, состояния приложения (в процессе выполнения / вывод результатов) и т.д. Естественно покрыть тестами
