Домашка от 2019.09.27

В исходниках примеры в папке ./examples с последнего занятия

  • Изучить синтаксис scss. Чтобы использовать его вам понадобится Node.js. После установки у вас появится пакетный менеджер npm в консоли. С ним подробнее познакомимся на js занятиях. Далее, следуя руководству, в консоли npm install -g sass. После этого у вас появится sass в консоли. Далее используем sass --watch input.scss output.css с указанием файлов ввода и вывода. Файл вывода подключается в html страницу через link.
  • Почитать ещё глубже о flexbox
  • (опционально) сверстать любой макет или хотя бы его часть и получить от меня огромный фидбек, где мы разберем по полочкам все ошибки
