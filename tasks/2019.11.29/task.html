Проект

  • Установить json-server в проект
  • Перенести вопросы из вашего кода в отдельный json файл
  • На основе примера в examples реализовать CRUD операции админки с json файлом
  • При создании и редактирования вопроса форма должна переиспользоваться
  • Перед удалением вопроса вы должны показать confirm окно бутстрапа
  • После успешной CRUD операции используйте редирект location.href на главную страницу со списком вопросов
  • Паблик проекта также должен использовать вопросы этого json файла
