    • Сделать тултип на чистом css. Да, всего одно задание. Да, для него надо знать очень много теории.

      Что понадобится

      • display: none для сокрытия тултипа
      • псевдокласс :hover для отмены display: none при наведении
      • позиционирование через position: absolute и transform: translate
      • Немного CSS3 через border-radius и box-shadow
      • сделать треугольник под тултипом. Для этого понадобится псевдоэлемент :after. В остальном, его также позиционируем, как описано ранее. Это обычный квадратный блок, который повернут на 45 градусов через transofrm: rotate
