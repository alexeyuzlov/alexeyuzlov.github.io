Домашка от 2019.10.18

Что почитать?

  • Функциональное программирование на JS (в 6 частях)
  • Написать пагинацию вида search(items: Article[], currentPage: number, pageSize: number, filterFn: Function sortFn: Function) для поиска по приложенному в examples json файлу статей. Скопируйте json содержимое внутрь вашего script.js файла как var db = [...]
  • В самом простом случае достаточно будет реализовать search(items, currentPage, pageSize)
  • (опционально) Реализация параметров filterFn и sortFn является опциональной, если кто-то хочет лучше разобраться с предикат функциями высшего порядка, которые рассматривались на последнем занятии. Это функции, которые позволяют применять пользовательские фильтры (например по массиву тегов) и сортировки (по дате создания по убыванию / возрастанию). Учтите, что сначала статьи должны быть отсортированы по дате и отфильтрованы по tags, только потом применять slice для выбора pageSize статей.
  • Тесты обязательны Имейте ввиду, что для тестов вы должны использовать свой собственный json заглушку, вместо предоставленного мной, достаточно прогонять для pageSize: 2, где в вашем массиве статей будет всего 5 записей
    • нормальные случаи, когда функция отрабатывает корректно
    • корректность входных параметров
    • корректность выходных данных
    • обработка ситуации что для заданной currentPage, pageSize может вернуться пустой массив
    • pageSize должен быть положительным
