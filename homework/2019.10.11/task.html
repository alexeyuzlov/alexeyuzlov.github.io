Что почитать?

  • Основы JS

Написать функции и unit-тесты к ним на фреймворке Jasmine. Заготовка для описания тестов и самих функций в папке examples. Для всех функций обрабатывать граничные условия и возвращать null если входное значение некорректно для функции (например передача undefined или null)

  • yesOrNo(value) Возвращает "Yes", "No" в зависимости от переданного value: boolean.
  • counter(total) Функция-счётчик. Возвращает значение result = 1..9 для чисел 1..9. При передаче total: number больше девяти возвращает "9+".
  • calculateAge(date) Возвращает положительное значение возраста в зависимости от переданной даты типа Date.
  • getRandomItem(list) На входе массив list произвольного типа. Возвращает произвольный элемент из заданного массива.
